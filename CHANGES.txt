Release v1.0.4, xx.xx.2026.

- Bug fixes and new features.
- Implemented new functions in the geography module. Added the required functionality because the 3D map uses ellipsoid height instead of Mean Sea Level (MSL).
- Added a class for forking processes with pseudoterminal file descriptors.
- Implemented tableprint to display data in a plain-text table and optionally export it as a LaTeX table.
- Implemented requireWasm and compileWasm for WebAssembly.
- Added search to the workspace panel.
- Implemented better vector and matrix display.
- Added search history to the editor search.
- Implemented search and replace for all open scripts.
- Implemented a symbol input panel in the editor.
- Enabled saving and loading of objects in the workspace.
- Implemented a variable inspector.
- Implemented table support.
- Reworked internal assignment and calling.
- Implemented tests.

Release v1.0.2, 13.03.2025.

- Bug fixes and new features.
- Added a source code book generator.
- Added a command window submit button.
- Implemented truncation of long strings in objects for display.
- Adapted the Editor for C++ Arduino code.
- Implemented an option to open files in the Editor on click (used to point to errors in C++ code).
- Removed the dependency on WMIC (fix for Windows 11).
- Added a serial-device module and implemented a Serial Terminal with a port and baud-rate selection dialog.
- Added an initial module for presentations.
- Added an initial module for mechanics.
- Added functions gridGradient and gridData.

Release v1.0.1, 16.02.2025.

- Bug fixes and new features.
- Added dispMonospaced function.
- Fixed version display.
- Parallel worker setup and context assignment now run only once.
- Fixed examples unpacking and added openExamplesFolder function.
- Added documentation indicating that the .JSL extension is used for scripts.
- Extended meshgrid to N dimensions.
- Extended reshape to use _ for automatic length.
- Added an example for the sensor reader.
- Changed window-closing behavior.
- Added multiple helper functions for windows and figures.
- Added adcToNewtons function.
- Added functions to open DevTools.
- Added searchHelp function.
- Added search to HTML documentation.
- Added installModule function.
- Added basic support for SISO system analysis with the control module (Computer-Aided Control System Design, CACSD).
- Added basic support for optimization with the optim module, based on the npm package fmin.
- Added support for the Real-Coded Mixed-Integer Genetic Algorithm.
- Added the Search Space Algorithm.
- Added support for 2D and 3D maps (functions webmap and geoglobe).
- Added support for gamepads using the Web Gamepad API.
- Added the functions checkArduino, compileArduino, and uploadArduino.
